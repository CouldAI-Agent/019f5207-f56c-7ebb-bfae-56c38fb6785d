# Software Developer Portfolio

A modern, responsive, single-page portfolio website for a software developer, built with Flutter.

## Features
- **Responsive Design**: Adapts gracefully to both mobile and desktop screens.
- **Modern Dark Theme**: A sleek, dark user interface that highlights content.
- **Portfolio Sections**: Includes Home, About, Skills, Projects, and Contact sections.
- **Single-Page Navigation**: Smooth scrolling or direct routing (depending on implementation) to navigate between sections.

## Tech Stack
- Flutter
- Dart

## Getting Started

1. Ensure you have [Flutter installed](https://flutter.dev/docs/get-started/install).
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app. It works best on Web or Desktop platforms for the full portfolio experience, but also supports mobile layout.
